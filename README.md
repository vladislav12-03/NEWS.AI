# NEWS.AI

Experimental Telegram news aggregator with AI-generated summaries.

## Status

⚠️ Project frozen / no longer actively maintained.

## Features

- Collects news from Telegram channels via Telethon
- Generates AI summaries
- News caching system
- Region selection
- Telegram bot interface
- Administrative tools

## Tech Stack

- Python
- python-telegram-bot
- Telethon
- Groq API
- JSON caching
- AsyncIO

## Why the project was stopped

The main goals of the project were achieved:

- learning Telegram Bot API
- experimenting with AI integrations
- working with asynchronous Python
- building a multi-module application

Development was stopped because the project served its educational purpose and interest shifted to other ideas.

## Disclaimer

This project is published as a portfolio and learning project. Some parts of the codebase may be experimental and unfinished.
